Project News

  • Jan 2019: BabyRobot project concludes showcasing how portable robotic setups can be used in schools for special education children - for details see the BabyRobot website
  • Mar 2018: BabyRobot project showcases some fascinating new use-cases for child-robot communication - for details see the BabyRobot concept videos
  • Jan 2016: BabyRobot project starts with the kickoff at NTUA, Athens, Greece. For more information see the BabyRobot website
  • Dec 2015: SpeDial project concludes producing exciting results for hot-spot detection and root-cause analysis in spoken dialogue systems also receiving excellent reviews!
  • Apr 2014: updated multilingual data for grammar induction tasks (including ontologies) are now available at the the PortDial website.

  • Dec 2014: kick-of for the EU-IST SpeDial project on spoken dialogue analytics in Luxembourg

Awards and Competitions

  • Feb 2018: The NTUA-SLP team wins 1st place in the multi-class twitter emotion classification task at SemEval 2018, 2nd and 3rd place at various other tasks
  • Jan 2016: Elected to the grade of IEEE fellow (class of 2016) for "contributions to human-centered speech and multimodal signal analysis."
  • Feb 23, 2016: The NTUA-USC team wins (1st place among 19 groups in) the tweeter sentiment analysis task 4-B (topic) SemEval 2016 competition using semantic-affective model adaptation techniques. Congrats to Elissavet and the team!
  • Sep 18, 2014: The USC SAIL team wins the Cognitive Load Sub-Challenge at the Speaker Trait Challenge at Interspeech 2014. Way to go Maarten and team!
  • Apr 30, 2014: The USC SAIL team scores 2nd place in the SemEval 2014 sentiment analysis competition (twitter sarcasm sub-task) using semantic-affective models.
  • Dec 1, 2013: The PortDial sponsored SemEval 2014 grammar induction task is up and running. For more details see the official SemEval 2014 website and the PortDial website.
